# Fördjupningsprojekt

## Beskrivning
Detta projekt är ett spel där användaren ska gissa ett slumpmässigt tal mellan 1 och 100.

## Funktioner
- Slumpar fram ett tal
- Räknar antal försök
- Ger ledtrådar
- Har felhantering om användaren skriver fel

## Programmering
Programmet använder:
- Funktioner
- Loopar
- Villkor
- Felhantering
- Moduler

## Hur programmet körs
Kör filen `main.py` i Python.
